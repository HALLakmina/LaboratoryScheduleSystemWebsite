## Installation & Run
1. Setup and select the place to clone the project:
    - Download and install [Node.js](https://nodejs.org/en/download)
    - Download and install [git-bash](https://git-scm.com/downloads)
    - Download and install [XAMPP](https://www.apachefriends.org/download.html)
    - Go to this folder location `xampp\htdocs\` (xampp install location) and open git-bash or CMD in that location.
2. Clone the repository:
    type flowing command in CMD or git-bash
   ```bash
   git clone https://github.com/HALLakmina/Inventory_Management_System.git
   cd Inventory_Management_System
   git fetch
   git pull
   cd admin
   cd admin-panel-backend
   npm install
   npm start run
   cd ..
   cd admin-panel-frontend
   npm install
   npm start run
   ```
3. open the XAMPP and start Apache and MySQL servers.

## Features
- Student and lecturers can identify lecher time years wise and free time.
- lecturers can request extra lecher time and cancel the schedule lecher.
- lab assistor can manage lab in his siting place communicate with lecturers.
- Give e-mail alert about lab-status for lecturers and student.
